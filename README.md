Eat Together
---

[English version](README_en.md)

Eat together是一个开源的基于Express与MongoDB的Web App，用以解决公司或者学校内基友们的点餐问题。

## 安装运行

- 首先需要安装Node.js和MongoDB
- `git clone https://github.com/FantasyMedia/EatTogether.git`
- `npm install && npm start`
